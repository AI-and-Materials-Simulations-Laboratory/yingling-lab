flory_fox_scraper
Python · Machine Learning

Scrapes glass transition temperature (Tg) data for polymers and performs Flory-Fox fitting. Enables data-driven prediction of polymer thermal properties from literature.

J. S. Peerless, N. J. B. Milliken, M. D. Manning, T. J. Oweida, Y. G. Yingling
Advanced Theory and Simulations (2018) · DOI: 10.1002/adts.201800129
GitHub DOI
lammps-mspin
C++ · LAMMPS Plugin

LAMMPS plugin for atomistic-resolution simulation of magnetic nanoparticles. Implements Zeeman alignment and dipolar assembly physics for spin-active systems in molecular dynamics.

Akhlak U. Mahmood & Yaroslava G. Yingling
ACS J. Chemical Theory & Computation (2022) · DOI: 10.1021/acs.jctc.1c01253
GitHub DOI
X3DBio2
Visualization · Analysis

Visual analysis tool for biomolecular structure comparison. Enables interactive 3D comparison of molecular conformations, supporting structure-function analysis workflows.

H. Yi, S. Thakur, L. Sethaphong, Y. G. Yingling

Visual analysis tool for biomolecular structure exploration. Predecessor to X3DBio2, providing interactive 3D visualization and structural analysis of biological macromolecules.

Database
RNAJunction

A curated database of RNA junctions and kissing loops for three-dimensional structural analysis and RNA nanodesign. Provides annotated structural data to support computational prediction of RNA tertiary structure and the design of RNA-based nanomaterials.

Patents

US010035995 B2 WO2017100240 A1 2018
CALB Variants — Lipase with Improved Synthesis Activity
K. Clendennen, Y. G. Yingling, H. Kim

Amino acid sequence variants of a lipase with improved activity for catalyzing synthesis reactions. Methods include predicting amino acid sites for change based on computational models of protein structure in non-aqueous conditions. Described enzyme variants show a 3- to 9-fold improvement in synthesis activity over the parent protein sequence.
